# Kubernetes CKA Practice Notes

This repository contains my personal notes and practice material created while preparing for the Certified Kubernetes Administrator (CKA) certification.

The content is based on hands-on labs and exercises completed on:

- Linux Foundation training
- KodeKloud Kubernetes labs
- killer.sh CKA exam simulators

## Topics

- Core Kubernetes concepts
- Scheduling
- Logging and monitoring
- Application lifecycle
- Networking
- Storage
- Security
- Cluster installation and maintenance
- Package management
- Troubleshooting

## Technologies

Kubernetes • Docker • Containers • DevOps • Cloud-Native Infrastructure

## Goals
- reinforce Kubernetes fundamentals
- document troubleshooting techniques
- provide real command examples
- create a knowledge base useful for DevOps roles
